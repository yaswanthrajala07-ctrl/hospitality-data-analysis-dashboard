# Hospitality-data-analysis-dashboard
Data analysis and dashboard project using Power BI, SQL, and Excel

## 📊 Project Overview
This project focuses on analyzing hospitality data to track revenue, occupancy, and booking performance using Power BI, SQL, and Excel.

## 🛠 Tools Used
- Power BI
- SQL
- Advanced Excel

## 📌 Key Features
- Data cleaning and preprocessing
- Data modeling using fact and dimension tables
- KPI metrics: Occupancy %, ADR, RevPAR, Cancellation %
- Interactive dashboards with filters and drill-down

## 📈 Insights
- Identified high cancellation rates in specific booking platforms
- Analyzed seasonal trends in occupancy and revenue
- Highlighted underperforming hotels and areas for improvement
